## Quick Start

### 1. Install Dependencies

```bash
cd stock-market-dashboard
pip install -r requirements.txt
```

### 2. Start the Server

```bash
cd stock-market-dashboard
python -m uvicorn backend.main:app --reload --port 8001
```

### 3. Open Dashboard

Navigate to http://localhost:8001 in your browser.

### 4. Configure Telegram

1. Get API credentials from https://my.telegram.org
2. Go to Settings in the dashboard
3. Enter API ID, API Hash, and phone number
4. Verify with the code sent to Telegram

### 5. Add Sources

1. Go to Sources tab
2. Click "Add Source"
3. Enter channel name and username (e.g., @stocktips)
4. Click "Fetch" to pull messages

### 6. Generate Recommendations

1. Select a time range using shortcuts or calendar
2. Click "Fetch News" to get latest market news
3. Click "Analyze & Recommend" to generate recommendations

## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/health` | GET | Health check |
| `/api/telegram/login` | POST | Telegram auth |
| `/api/telegram/verify` | POST | Verify code |
| `/api/sources` | GET/POST | Manage sources |
| `/api/messages` | GET | Get signals with filters |
| `/api/news` | GET | Get market news |
| `/api/news/fetch` | POST | Fetch RSS feeds |
| `/api/stocks/indices` | GET | Get NIFTY/SENSEX |
| `/api/analyze` | POST | Analyze and recommend |
| `/api/recommendations` | GET | Get recommendations |
